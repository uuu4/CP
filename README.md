## overview

this repo collects solutions written while practicing on cses, codeforces, atcoder, and codechef. each solution lives as a standalone `.cpp` file, grouped by the platform it came from.

## folders

| folder | platform | link |
|---|---|---|
| `cses/` | cses problem set | [cses.fi](https://cses.fi/problemset/) |
| `codeforces/` | codeforces | [codeforces.com](https://codeforces.com) |

## running a solution

```bash
g++ -std=c++17 -O2 -o sol path/to/solution.cpp
./sol < input.txt
```
